    # Attributes

    参数 说明 类型 是否必要 默认值
    category 分类数据 Array true -

    # Events

    事件名称 说明 回调参数
    parent_click 点击每个父分类数据的回调，传参返回label和value，可以根据返回的参数进行其他操作 {"label":"豪华电视","value":"7914"}
    child_click 点击每个子分类数据的回调，传参返回label和value，可以根据返回的参数进行其他操作 {"label":"环保音响","value":"6767"}
